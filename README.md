# TeachUA Infrastructure

## 🚀 Overview
Cloud-native infrastructure and automated CI/CD pipeline for the TeachUA platform. 

## 🛠 Tech Stack
* **Cloud:** Microsoft Azure
* **IaC:** Terraform (Modular architecture)
* **Orchestration:** Kubernetes (v1.27+)
* **CI/CD:** GitHub Actions
* **Security:** Trivy (Image scanning), SonarCloud (SAST)
* **Observability:** Grafana, Loki, Splunk, Jaeger

## 🏗 Key Features
* **Zero-Trust Security:** Secret management via Azure Key Vault.
* **Scalability:** Horizontal Pod Autoscaling readiness.
* **Observability:** Full-stack distributed tracing and centralized logging.

> **Note:** Production environment (teacherio.pp.ua) was successfully validated and then decommissioned to optimize cloud resource costs.
